# Python Script

This folder contains the Jupyter Notebook scripts used to process and generate the Large CheckFile report.

## Files

- **Large Check File 08th June-Copy1.ipynb** — Main Jupyter Notebook that processes the Large CheckFile shipment data. It covers end-to-end logistics tracking including order details, seller info, hub movement, forward/RTO status, delivery ageing, OFD attempts, FDP status, and logistics breach analysis.

## Requirements

- Python 3
- Jupyter Notebook / JupyterLab
- pandas, numpy (for data processing)

## Usage

1. Place the required input data files and mapping files in the appropriate directories.
2. Open the `.ipynb` file in JupyterLab or Jupyter Notebook.
3. Run all cells sequentially to generate the CheckFile report output.
