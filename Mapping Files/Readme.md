# Purpose

These mapping files are used by the Python scripts to enrich and validate CheckFile shipment data — including zone classification, hub assignment, and seller/vendor mapping — to generate accurate end-to-end logistics reports.

## Mapping Files

This folder contains the mapping and reference files used in the CheckFile report pipeline.

## Files

- **Large XD mapping.xlsx** — Maps large shipment XD (cross-dock) data for routing and hub assignment.
- **Rev 16May.xlsx** — Revision mapping file updated as of 16th May, used for reference data corrections.
- **Zone Mapping.xlsx** — Contains zone-level mapping for origin and destination pincodes used in logistics processing.
